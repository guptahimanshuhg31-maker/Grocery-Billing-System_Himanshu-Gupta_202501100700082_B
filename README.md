## Description
This project is a simple Python-based Grocery Store Billing System.
It calculates the total cost of 5 items based on price and quantity.
A 10% discount is applied if the total bill exceeds Rs. 100.

## Features
- Accepts price for 5 items
- Accepts quantity for each item
- Calculates total amount
- Applies 10% discount if total > Rs. 100
- Displays original total, discount, and final payable amount
